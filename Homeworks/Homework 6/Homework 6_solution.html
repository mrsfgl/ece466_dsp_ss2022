[20] Determine if the Fourier Transform $X(\omega)$ of a signal $x[n]$ is a) even or odd, and b) imarinary or real when $x[n]$ is:

  1. Real and even.
  2. Imaginary and even.
  3. Real and odd.
  4. Imaginary and odd.

Show your work.

Answer for 1.1:

  1. $x[n] = x_R[n]$, where $x_R[n]$ is the real part of $x[n]$. Equivalently, $x_I[n]=0$, where $x_I[n]$ is the imaginary part of $x[n]$.
  2. We know that $$X(\omega) = \sum_{n=-\infty}^{\infty} \left(x_R[n]\cos(\omega n)+x_I[n]\sin(\omega n)\right)+j\left( -x_R[n]\sin(\omega n)+x_I[n]\cos(\omega n)\right).$$
  3. The sum of an odd function from $-\infty$ to $\infty$ is zero. Thus, $$\sum_{n=-\infty}^{\infty}-x_R[n]\sin(\omega n)=0$$
  4. Thus, $X(\omega) = \sum_{n=-\infty}^{\infty} x_R[n]\cos(\omega n).$ Thus, $X(\omega)$ is a sum of a real signal, hence it is always real.
  5. Also, $X(-\omega) = \sum_{n=-\infty}^{\infty} x_R[n]\cos(-\omega n) = \sum_{n=-\infty}^{\infty} x_R[n]\cos(\omega n) = X(\omega)$, so it is even.

Answer for 1.2:

Now, we only have $x_I[n].$ so, following a similar analysis as 1.1 (as the $x_I[n]\sin(\omega n)$ is odd), we end up with $$X(\omega) = \sum_{n=-\infty}^{\infty} jx_I[n]\cos(\omega n)$$

  1. Since this is the sum of purely imaginary numbers, $X(\omega)$ is imaginary.
  2. Similar to 1.1, $X(\omega)= X(-\omega),$ hence it is even.

Answer for 1.3:

Now, since $x_R[n]$ is odd, $x_R[n]\sin(\omega n)$ is going to be even. Hence: $$X(\omega) = \sum_{n=-\infty}^{\infty} -jx_R[n]\sin(\omega n)$$

  1. Since this is a sum of purely imaginary numbers, $X(\omega)$ is imaginary.
  2. Also, $X(-\omega) = \sum_{n=-\infty}^{\infty} -jx_R[n]\sin(-\omega n) = \sum_{n=-\infty}^{\infty} jx_R[n]\sin(\omega n) = -X(\omega)$, so it is odd.

Answer for 1.4:

Similar to 1.3, we will be left with only $x_I[n]\sin(\omega n)$. $$X(\omega) = \sum_{n=-\infty}^{\infty} x_I[n]\sin(\omega n)$$

  1. Since this is a sum of real numbers, $X(\omega)$ is real.
  2. Following 1.3, $X(\omega)$ is odd.

# Code for question 4

def dtft(x, n, w):
    """ DTFT computed at w

    Parameters:
    ---
        x : finite duration sequence over n
        n : sample position vector
        w : frequency location vector

    Returns:
    ---
        X : DTFT values computed at frequencies w
    """
    N = len(n)
    X = np.zeros_like(w)
    for i in range(N):
        X = X + x[i]*np.exp(-1j*n[i]*w)

    mag_X = np.abs(X)
    angle_X = np.angle(X)
    fig, (ax1, ax2) = plt.subplots(2,1, sharex=True)
    ax1.plot(w, mag_X/mag_X.max())
    ax1.set_title('Magnitude')
    ax1.grid()
    ax2.plot(w, angle_X)
    ax2.set_title('Phase')
    ax2.set_xlim(-np.pi, np.pi)
    ax2.grid()

    return X
